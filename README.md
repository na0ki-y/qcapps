# Installation :balloon:
`python(stremlit)`で実装されています。
以下の通り、`pipenv`で環境構築後、ローカル環境で実行できます。

```bash
git clone git@github.com:na0ki-y/qcapps.git
cd qcapps
pipenv install
streamlit run app.py
```
